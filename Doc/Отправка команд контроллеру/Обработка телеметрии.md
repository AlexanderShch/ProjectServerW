## ?? Как это работает на стороне сервера:

```
Контроллер отправляет данные
          |
    ClientHandler (SServer.cpp)
    получает ВЕСЬ входящий трафик
          |
    Проверяет первый байт ответа
          |
    -------------
    |           |
Type=0x00   Type=0x01-0x04
Телеметрия  Ответ на получение команды
    |           |
AddData()   EnqueueResponse()
              |
        ResponseQueue
              |
      ReceiveResponse()
      WaitOne(timeout)
```
