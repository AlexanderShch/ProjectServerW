# Приём и обработка пакетов (сервер, UTF-8)

Ключевые форматы:

1. Входящий кадр (контроллер -> сервер по TCP):  
   `[AA 55][Type][Cmd][Status][DataLen][Data...][CRC16]`

2. Телеметрия: `Type=0x03`, `Cmd=0x08`, `Status=0x00`, `DataLen=45`.

3. Лог алгоритма: `Type=0x00`, `Cmd=0x01`, `Status=0x00`, `DataLen=89`.

4. Исходящая команда сервера (без `AA 55`):  
   `[Type][Code][DataLen][Data...][CRC16]`
